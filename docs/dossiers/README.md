# Design Dossiers

This directory contains the canonical "Design Dossiers" for the factory.

## Format
All dossiers must be named `DOS-<YYYYMMDD>-<slug>.md` and adhere to the [Design Dossier Template](../pm/DESIGN_DOSSIER_TEMPLATE.md), including strict YAML front-matter validated by [schema](../schemas/design_dossier.schema.json).

## Governance
- **No Spec without a Dossier**: Specs are manufactured from approved dossiers.
- **Owner**: Nexus Agent.
